# Kick Chat

Simple app to see your Kick/Twitch chat while playing games.

Useful for streamers with only one monitor.

Set your game display mode to **Borderless** for the overlay to work.
- May not work with some games.
- Exclusive fullscreen is not supported.

Thanks to **Corard** for: https://kick-chat.corard.tv/

## Features

- Kick chat overlay
- Twitch chat overlay
- Discord live alerts (webhooks)
- Telegram live alerts
- OBS WebSocket support
- Basic overlay customization (fonts, colors, opacity, borders, etc.)

## OBS Setup

1. Open **OBS Studio**.
2. Go to **Tools → WebSocket Server Settings**.
3. Enable the WebSocket server.
4. Enter the Host, Port, and Password in Kick Chat.
